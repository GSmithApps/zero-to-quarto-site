---
title: "Binary Vs. Plain Text"
marp: true
class: invert

---

- Can open plain text files in a text editor (
  and manually modify them), but you can't with binary
  files
- Binary is more compressed
- Can version control plain text files

---

# Examples
- plain text
  - markdown
  - python files
  - txt file
  - CSV files
  - json files
- binary
  - excel
  - word
  - parquet
